# 【FizzBuzz問題】新しい言語に挑戦する時用の基礎文法確認

上記に対する成果物  

## 要件

- 3の倍数の場合は「Fizz」と出力。
- 5の倍数の場合は「Buzz」と出力。
- 3の倍数かつ5の倍数は「FizzBuzz」と出力。
- いずれでもない場合はその数を出力。

上記条件で、1から100までの数字において、標準出力に改行つきで出力する。

## 実行環境

ruby 2.7.1

## 実行方法

    git clone https://github.com/amano-hiiragi/techcommit_fizzbuzz_01
    cd techcommit_fizzbuzz_01
    ruby fizzbuzz.rb
